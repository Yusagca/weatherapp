# Weather App

## Description (Açıklama)

- **English**:  
  The Weather App is a React-based application that provides real-time weather updates using the [Open-Meteo API](https://open-meteo.com/). The app allows users to search for a city's weather conditions, including temperature, precipitation, and wind speed, while displaying a dynamic video background based on weather conditions.

- **Türkçe**:  
  Hava Durumu Uygulaması, [Open-Meteo API](https://open-meteo.com/) kullanarak gerçek zamanlı hava durumu güncellemeleri sağlayan React tabanlı bir uygulamadır. Kullanıcılar, bir şehrin hava durumu koşullarını (sıcaklık, yağış, rüzgar hızı gibi) arayabilir ve hava durumuna göre değişen dinamik bir video arka planıyla görüntüleyebilir.

---

## Features (Özellikler)

- **English**:  
  - Real-time weather updates for any location.
  - Dynamic video backgrounds based on weather conditions (e.g., clear, rainy, snowy).  
    **Note**: Due to Vercel integration limitations, video backgrounds are not functioning on Vercel. Below, you can find visuals demonstrating the full project. 
  - Displays 3-day weather forecast (highest and lowest temperatures).
  - Search functionality for any city worldwide.
  - User-friendly and responsive design.
  - Social media links (GitHub, LinkedIn, Instagram).

- **Türkçe**:  
  - Herhangi bir konum için gerçek zamanlı hava durumu bilgileri.
  - Hava koşullarına göre dinamik video arka planları (örneğin, açık hava, yağmurlu, karlı).  
    **Not**: Vercel entegrasyonundaki kısıtlamalar nedeniyle, arka plan videoları Vercel üzerinde çalışmamaktadır. Aşağıda, tam projeyi gösteren görseller bulunmaktadır.
  - 3 günlük hava durumu tahmini (en yüksek ve en düşük sıcaklıklar).
  - Dünya genelinde şehir arama özelliği.
  - Kullanıcı dostu ve duyarlı tasarım.
  - Sosyal medya bağlantıları (GitHub, LinkedIn, Instagram).

## Usage (Kullanım)

- **English**:
  1. Open the app using this link:  
     [https://weatherapp-blond-two.vercel.app/](https://weatherapp-blond-two.vercel.app/)
  2. Enter the name of the city you want to search in the search bar.
  3. Click the search icon to fetch weather information.
  4. View real-time weather updates, including temperature, precipitation, and wind speed.
  5. Explore the 3-day forecast for the selected location.

- **Türkçe**:
  1. Uygulamayı şu link üzerinden açın:  
     [https://weatherapp-blond-two.vercel.app/](https://weatherapp-blond-two.vercel.app/)
  2. Arama çubuğuna aramak istediğiniz şehrin adını girin.
  3. Hava durumu bilgilerini getirmek için arama simgesine tıklayın.
  4. Sıcaklık, yağış miktarı ve rüzgar hızı gibi gerçek zamanlı hava durumu bilgilerini görüntüleyin.
  5. Seçilen konum için 3 günlük hava tahminini keşfedin.

## Technologies Used (Kullanılan Teknolojiler)

- **React**: Frontend library for building the user interface.
- **Open-Meteo API**: Provides weather data for any location.
- **TailwindCSS**: Utility-first CSS framework for styling.
- **React Icons**: Collection of popular icons for UI components.
- **Dynamic Video Backgrounds**: Enhances the user experience with weather-based visuals.  
  **Note/Not**: Due to Vercel integration limitations, video backgrounds are not functioning on Vercel.
